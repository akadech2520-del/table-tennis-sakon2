# ระบบสมาชิกชมรมกีฬาเทเบิลเทนนิส สพป.สกลนคร เขต 2

ชุดไฟล์นี้รวมหน้าเว็บไซต์ ระบบสมาชิก ภาพกิจกรรม บอร์ดถาม–ตอบ และหน้าผู้ดูแลระบบไว้เรียบร้อยแล้ว

## ไฟล์

- `index.html` — หน้าเว็บไซต์
- `style.css` — รูปแบบและการแสดงผลบนมือถือ
- `app.js` — การทำงานของหน้าเว็บและการเชื่อม API
- `Code.gs` — Backend สำหรับ Google Apps Script, Google Sheets และ Google Drive

## ทดลองบนเครื่อง

เปิด `index.html` ผ่านเว็บเซิร์ฟเวอร์ในเครื่อง ระบบจะเข้าโหมดสาธิตและเก็บข้อมูลใน `localStorage`

บัญชีสาธิต:

- ชื่อผู้ใช้: `admin`
- รหัสผ่าน: `pingpong2569`

โหมดสาธิตจะไม่ส่งข้อมูลไปยัง Google Sheets

## ติดตั้ง Backend

1. เปิด Google Sheet ที่ต้องการใช้ แล้วเลือก **Extensions > Apps Script**
2. นำเนื้อหาใน `Code.gs` ไปวาง
3. ตรวจสอบ `CONFIG.SPREADSHEET_ID` ให้ตรงกับ Google Sheet
4. รัน `setupSystem()` หนึ่งครั้งและอนุญาตสิทธิ์
5. เปลี่ยนชื่อผู้ใช้และรหัสผ่านใน `setupAdminAccount()` แล้วรันหนึ่งครั้ง
6. เลือก **Deploy > New deployment > Web app**
7. ตั้งค่า **Execute as: Me** และ **Who has access: Anyone**
8. คัดลอก URL ที่ลงท้ายด้วย `/exec` ไปแทนค่า `GAS_API_URL` ใน `app.js`
9. นำ `index.html`, `style.css` และ `app.js` ขึ้นโฮสต์เว็บไซต์

## หมายเหตุสำคัญ

- หลังแก้ `Code.gs` ต้องสร้าง Deployment เวอร์ชันใหม่
- ควรเปลี่ยนรหัสผ่านเริ่มต้นก่อนเปิดใช้งานจริง
- หากโดเมน Google Workspace ไม่อนุญาตแชร์ไฟล์แบบ “Anyone with the link” ภาพกิจกรรมอาจไม่แสดงต่อบุคคลภายนอก ต้องปรับนโยบายแชร์ไฟล์หรือใช้ที่เก็บภาพอื่น
- `setupSystem()` ในชุดนี้ไม่ล้างข้อมูลเดิม แต่ควรสำรอง Google Sheet ก่อนอัปเดตระบบทุกครั้ง
